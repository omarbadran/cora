== Description ==

**[Cora](https://plugincube.com/?utm_source=wp-repo&utm_medium=link&utm_campaign=readme)** Is the ultimate WordPress admin theme & white label plugin designed to help you achieve a modern clean look that is harmonious with your brand. packed with tons of options to help you customize every aspect of the admin area & login page.

If you like Cora, please consider leaving a review here on WordPress. Thank you!

### Admin Menu
* Add logo
* Reorder menu items
* Hide items for specific users
* Change item title
* Change item icon
* Add links
* Add custom pages

### Toolbar
* Hide default items (WordPress logo, Comments ...etc)
* Add custom links

### General
* Add custom dashboard widgets
* Change footer text
* Show useful information about your WordPress Installation in footer
* Show post image column in posts list
* Hide Screen Options buttons

### Optimization
* Hide WP Version Meta-Tag
* Disable WP emoji
* Disable RSS Feed
* Disable Pingback
* Enable SVG & ICO file support
*And many more options ... Try it yourself :)

### Theme
Customize every part of your admin theme visually with live preview. Change the design to represent your brand or personal taste. You can choose a pre-designed theme or make your own. you can also enable dark theme.

### Custom Login Page
Differentiate your website with a unique modern login & sign-up page. Upgrade to customize & rebrand the login page completely within seconds.



== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
